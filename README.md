# Chat App

A ipfs based chat application, meant to be e2e encrypted while also being accessible from multiple devices at any point of time.

## Devs - (Team Pseudo-Nerds)

- [Mustafa Officewala](https://github.com/Mustafa2856)

## Problem Statement

The current problem we feel is that most of the chat applications are either e2e encrypted like WhatsApp or either can be accessed on multiple devices concurrently like Slack, Discord.
The reason being, how Encryption is done with public-private key pair.

## Tech Stack

- CocroachDB
- Node.js
- Reactjs
- js-ipfs
